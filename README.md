# 対話式商品検索アプリ
Watsonと対話しながら商品を検索するアプリです。

## 開発言語
 - PHP

## 概要
 - IBMCloudを使用しています。
 - 対話はWatsonAPIを使用しています。
 - 商品検索は楽天APIを使用しています。

## 画面
　1. 質問に答えていくと楽天からおすすめの商品を検索して表示します。
